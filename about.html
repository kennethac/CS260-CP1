What is TaskBurner?

TaskBurner is a simple, intuitive and effective way to manage your class load.

Create a new project for each class or semester assignment, then add the tasks and steps for each thing you need to do. Then use TaskBurners organizational features to help you manage your semester and nail all of your assignments.

Currently it is just static and holds bogus tasks for my CS 260, Math 313 and English 316 classes - however it will be developed further.
